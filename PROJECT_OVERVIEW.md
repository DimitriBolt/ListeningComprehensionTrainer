## 🎯 Цель проекта

Создать приложение для тренировки **listening comprehension** английского языка с использованием AI.

**Ключевые особенности:**
- 🎙️ Голосовое управление (программа сама понимает, когда вы закончили говорить)
- 👨‍🏫 Учитель AI, который говорит медленно и простым языком
- 🔄 Бесконечный диалог для тренировки
- 📱 Поддержка desktop, web и мобильных приложений
