📋 ШАГИ:

1️⃣ ОТКРОЙТЕ ТЕРМИНАЛ И ПЕРЕЙДИТЕ В ПАПКУ:

   cd ~/PycharmProjects/ListeningComprehensionTrainer


2️⃣ АКТИВИРУЙТЕ VIRTUALENV:

   source .venv/bin/activate

   (В начале строки появится (.venv) - это означает, что всё работает)


3️⃣ ЗАПУСТИТЕ ПРИЛОЖЕНИЕ (выберите один из вариантов):

   ✅ ВАРИАНТ A: Интерактивный режим с меню (рекомендуется)
      python main.py

      Затем нажмите "1" для начала сеанса

   ✅ ВАРИАНТ B: Сразу начать сеанс обучения (быстрее)
      python main.py --practice

   ✅ ВАРИАНТ C: Посмотреть историю предыдущей беседы
      python main.py --history


4️⃣ ЧТО ДАЛЬШЕ:

   • Когда программа попросит - говорите в микрофон
   • Программа запишет ваше высказывание
   • Будет распознана речь (текст)
   • ChatGPT сгенерирует ответ учителя
   • pyttsx3 озвучит ответ (вы услышите голос!)
   • Введите "Y" для следующего раунда или "n" для выхода
   • После выхода нажмите Ctrl+C

════════════════════════════════════════════════════════════════════
🆘 ЕСЛИ ЧТО-ТО ПОШЛО НЕ ТАК:
════════════════════════════════════════════════════════════════════

"Микрофон не найден"
  → Проверьте подключение микрофона
  → Проверьте уровень громкости
  → Перезагрузитесь

"Whisper не распознаёт речь"
  → Говорите громче и четче
  → Убедитесь, что микрофон включен
  → Проверьте интернет-соединение

"TTS не озвучивает"
  → Проверьте громкость на компьютере
  → Проверьте, что динамики включены

"API ошибка"
  → Проверьте интернет-соединение
  → Проверьте баланс на счёте OpenAI
  → Проверьте OPENAI_API_KEY в .env

📚 УРОВНИ ЯЗЫКА:
════════════════════════════════════════════════════════════════════

A2 (Начинающий):
  • Простой словарь
  • Короткие предложения
  • Максимум 3 предложения в ответе
  • Пример: "Hello, my name is Dmitri. I like reading books."

B1 (Средний):
  • Средний словарь
  • Естественные предложения
  • Максимум 4 предложения в ответе
  • Пример: "I've been learning English for three years. It helps me
           communicate better with people around the world."

B2 (Продвинутый):
  • Сложный словарь
  • Различные грамматические структуры
  • Максимум 5 предложений в ответе
  • Пример: "The multifaceted nature of language acquisition necessitates
           a comprehensive approach encompassing both linguistic theory
           and practical application."
